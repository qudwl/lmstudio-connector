# lmstudio-connector

VS Code's Copilot Chat natively supports Ollama integration for local models. However, it does not support LMStudio's local models. This extension bridges the gap by allowing LMStudio models to be used through the LM Studio SDK and the VS Code Language Model Chat Provider API.

## Extension Settings

This extension contributes the following settings:

- `lmstudio.enable`: Enable/disable this extension.
- `lmstudio.apiKey`: Your LMStudio API key (if required).
